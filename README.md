# 📌 Counter Program

A simple interactive counter application built using **HTML, CSS, and JavaScript**.

---

## ✨ Features
- ➕ Increase counter  
- ➖ Decrease counter  
- 🔄 Reset counter  
- ⚡ Real-time updates  
- 🎨 Styled UI with hover effects  

---

## 🧠 What I Learned
- DOM manipulation using `getElementById()`  
- Handling events with `onclick`  
- Updating UI dynamically with `textContent`  
- Managing state with variables  

---

## 🚀 Purpose
This project is part of my journey to learn JavaScript and build a strong foundation in front-end development.
