# Simple-Counter-Program
A simple interactive counter application built using HTML, CSS, and JavaScript.
This project demonstrates basic front-end development concepts such as DOM manipulation, event handling, and dynamic UI updates.
✨ Features
Increase, decrease, and reset counter value
Real-time display updates
Clean and responsive UI
Styled with basic CSS and hover effects
🧠 What I Learned
Using getElementById() to access DOM elements
Handling user interactions with onclick events
Updating content dynamically with textContent
Managing state using variables
🚀 Purpose
This project is part of my journey to learn JavaScript and build a strong foundation in web development.
